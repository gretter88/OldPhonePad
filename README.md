# OldPhonePad

Desafío de codificación C# de Iron Software. Simula el comportamiento de un teclado de teléfono antiguo (tipo T9).

## Ejemplos de entrada y salida

| Entrada                  | Salida |
|--------------------------|--------|
| `33#`                     | E      |
| `227*#`                   | B      |
| `4433555 555666#`         | HELLO  |
| `8 88777444666*664#`      | TOO    |
